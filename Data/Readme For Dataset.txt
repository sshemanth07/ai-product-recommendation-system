Dataset Availability

The dataset used in this project is derived from the Synerise eCommerce dataset, which is publicly available online. Due to the large size of the dataset, it is not included directly in the repository.

A subset of the dataset used for experimentation can be accessed via the following link:
https://drive.google.com/drive/folders/1yuwVGWtauJOAbxT1gk2Nri8pxlY0cN9V?usp=sharing

Note: Access to the dataset is restricted and may require a UTS email account to view and download the files.
